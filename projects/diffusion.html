Diffusion for Molecule Generation

Generative GNN 2025 pytorch torch_geometric EGNN RDKit

Motivation

Molecule generation is a key challenge in drug discovery — finding new valid molecular structures efficiently is expensive with traditional methods. Diffusion models have shown strong results in audio and image generation; this project explores whether the same framework can produce valid 3D molecules. The QM9 dataset was chosen for its standard benchmark status and easy integration via torch_geometric.

Diffusion framework

The model follows a standard continuous diffusion setup, but applied to 3D atomic coordinates:

  • Forward process: Gaussian noise is progressively added to the 3D coordinates of each atom in the molecule, according to a noise schedule. After T steps, the coordinates are approximately standard Gaussian.
  • Reverse process: Starting from random noise, the model iteratively denoises coordinates back toward a valid molecular structure. At each step, a learned score network predicts the noise to subtract.

Score model — EGNN

The score network is an Equivariant Graph Neural Network (EGNN), which respects the 3D symmetries of molecules (rotations and translations). The model is conditioned on:

  • Atom features (the 11-dimensional feature vector per atom from QM9)
  • Atom types (data.z)
  • Graph adjacency (bonds between atoms)
  • Time embedding (which diffusion step we are at)

Evaluation

Generated molecules are evaluated on three criteria:

  • Validity: whether RDKit can parse and sanitize the molecule (Chem.SanitizeMol).
  • Overlapping positions: whether atoms are placed too close together in 3D space.
  • Realistic energy: whether the generated geometry has physically plausible energy.
