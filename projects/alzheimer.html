Alzheimer's Disease Classification using Graph Attention Networks and LSTMs

GNN Deep Learning Medical Jul 2025 pytorch torch_geometric GAT LSTM

Overview

The project classifies the severity of Alzheimer's disease into three distinct stages: Non-Demented, Very Mild Dementia, and Mild/Moderate Dementia. The dataset used is OASIS-1, containing roughly 80K 2D MRI brain slices (~61 slices per patient scan).

Rather than treating each slice as an independent image, the pipeline models the full 3D scan as a sequential volume, capturing the progressive structural changes that characterize Alzheimer's progression.

Pipeline

  • Preprocessing: Individual 2D MRI slices are segmented into superpixels using SLIC. Each superpixel becomes a graph node (features: mean intensity, variance, spatial centroid), and edges represent spatial adjacency between regions.
  • Graph Attention Network (GAT): A GAT processes each slice's graph independently, learning attention-weighted relationships between brain regions and producing a rich per-slice embedding.
  • LSTM: The sequence of ~61 GAT embeddings from a single patient scan is passed through an LSTM, which captures the progressive changes across the brain volume and outputs the final severity classification.

Handling class imbalance

The OASIS-1 dataset is heavily imbalanced — the Non-Demented class dominates. Oversampling via image duplication was impractical at scale, so class weights were computed as the inverse class frequency and passed to a weighted CrossEntropyLoss.

Results

Training accuracy reached approximately 67% after 11 epochs, with a steadily decreasing training loss. The model demonstrated that treating the MRI volume as a sequence (rather than independent slices) significantly helps capture the structural patterns associated with dementia progression.
