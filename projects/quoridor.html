Quoridor

Software Dev Feb 2026 java

Overview

A full implementation of the Quoridor board game in Java, developed as part of the Software Development course at the University of Trieste. Quoridor is a two-to-four player abstract strategy game where players move a pawn across a 9×9 grid while placing walls to block opponents.

What was implemented

  • Complete game logic: pawn movement rules, wall placement validation, path-existence checking (to ensure players can never be completely blocked), and win conditions.
  • Player AI: a computer-controlled opponent using a heuristic-based strategy to decide between advancing and placing walls.
  • Graphical UI: a visual board rendered in Java, with interactive click-to-move and wall placement.
  • Multi-player support: both human vs. human and human vs. AI modes.

Technical notes

The path-existence check after each wall placement uses a BFS/DFS traversal to guarantee the game rules (a player must always have a reachable path to the goal row) are respected. Object-oriented design was prioritized, separating game state, rendering, and AI logic into distinct components.
